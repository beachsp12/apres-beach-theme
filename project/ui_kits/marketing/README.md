# Après Beach — Marketing Site UI Kit

The brand's `.com` — homepage, lookbook, journal, and PDP. Built around photography and the two-color rule.

## Files
- `index.html` — interactive click-thru of the homepage with cross-links to lookbook + PDP
- `Nav.jsx` — sticky transparent → solid sand on scroll
- `Hero.jsx` — full-bleed editorial with protection gradient
- `EyebrowSection.jsx` — section header pattern (eyebrow + display headline)
- `ProductGrid.jsx` — image-led product cards
- `Editorial.jsx` — magazine-style 2up image + copy block
- `Footer.jsx` — navy footer, small mark, tracked caps

Drop into a sand background. All interactive states are tonal — no new hues.
