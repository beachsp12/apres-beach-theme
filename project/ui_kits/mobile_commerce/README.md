# Après Beach — Mobile Commerce App UI Kit

Native-feel iOS mobile commerce app. Three primary tabs (Shop, Lookbook, Bag) + an Account drawer.

## Files
- `index.html` — interactive click-thru: shop → PDP → bag → checkout
- `MobileNav.jsx` — bottom tab bar
- `MobileShop.jsx` — vertical product grid with filter chips
- `MobilePDP.jsx` — full-bleed image, sticky buy bar
- `MobileBag.jsx` — line items, totals, CTA
- `MobileLookbook.jsx` — full-screen swipeable editorial cards
